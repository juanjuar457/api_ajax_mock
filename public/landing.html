Order For Later v 1.0

Full Stack Capstone | Juan Juarez

This web app is an order form for the compounding pharmacy I work in, It replaces a white board where
each individual material/ingredient would have to be hand written.

Order for later solves the inconvience of the white board by allowing users to simply visit a link and input
their requested materials and have them stored on a database. Version 1 allows for entry of up to 100 items and
a delete button for any material added, everything added/ removed is active to the server!Order for later also
allows highlighting to indicate backorder of a particular materials with 1 click!

Future Features
  • Fully functional login / user registration system
  • Fully functional password system
  • storage for over 10,000 entries spanning over 2 years of entries
